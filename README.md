# abarichello/godot-ci

Docker image to export Godot Engine games. Templates for Gitlab CI/GitHub Actions deploy to GitLab/GitHub Pages or Itch.io

Hardened by [Chainguard](https://www.chainguard.dev) from the upstream action at [https://github.com/abarichello/godot-ci](https://github.com/abarichello/godot-ci).
